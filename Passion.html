PRÉSENTATION DE MA PASSION :

Description :

  • Ma passion, qui me tient particulièrement à cœur, est le GOLF.
  • Ce sport est pour moi un moyen de détente et d'amusement.
  • J'ai donc décidé d'écrire quelques informations importantes à ce sujet.

    • Le terrain :

    Un parcours de golf est souvent constitué de 18 trous qui suivent à peu près le même schéma :
  • - Le départ où se passe le premier tir
  • - Le fairway qui est entouré du rough et à la fin le green où se situe le trou.
  • - Le green sur lequel se trouve le trou
  • - Il peut y avoir des obstacles sur le terrain tels que des bunkers qui sont constitués de sable et les obstacles d'eau, il existe d'autres obstacles, mais ces deux-là sont les plus courants.
    		•

    Le matériel :

    Au golf, le matériel est un outil essentiel pour y jouer. Il est divisé en deux grandes catégories :
    Les Consommables :
    Les consommables sont tous les objets que nous devons changer régulièrement. Tels que :
  • Les balles,
  • Les gants,
  • Les tees...
    		•
    Les Clubs :
    Ils sont répartis en trois catégories qui sont :
  • - Le putter : il convient au jeu sur le green et apporte le plus de précision.
  • - Les Fers : ils correspondent au jeu sur le ferway et sur le rough pour la majorité du temps. Répartie en 2 catégories :
  •   *Les Wedges sont les clubs les plus ouverts pour tirer a une longueur plutôt faible, mais avec beaucoup de précision
  •   *Les autres fers permettent de tirer assez loin et plus le numéro qu fer est faible plus il permet d'avoir de la distance.
  • - Les Bois : 3 catégories :
  •   *Les hybrides sont la jonction entre les fers et les bois.
  •   *Les Bois permettent d'obtenir une longue distance et une légère possibilité de les jouer sur le fairway
  •   *Les Drivers, ils sont réservés presque exclusivement au départ et permettent de couvrir la plus grande distance, mais avec le moins de précision.
    • ATTENTION : il est interdit d'avoir plus de 14 clubs dans son sac !!!
